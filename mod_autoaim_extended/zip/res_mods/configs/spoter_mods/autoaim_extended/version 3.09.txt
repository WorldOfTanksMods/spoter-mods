Автор: spoter
Название: autoaim_extended
Текущая версия 3.09 для 0.9.12 (18/11/2015)

Описание:
Мод расширенной индикации состояния автоприцела
	Индикация захваченой автоприцелом цели
		1. В пределах прямой видимости включается подсветка по контуру
		2. Образуется полупрозрачное, цветное энергополе
		3. Если цель находится вне зоны видимости, за пределами экрана, появляется индикатор направления с дальностью до цели
	Цвет индикации можно менять в конфигурационном файле.

	Варианты цветов:
		синий	blue
		коричневый	brown
		шоколадный	chocolate
		васильковый	cornflower_blue
		кремовый	cream
		голубой	cyan
		изумрудный	emerald
		золотой	gold
		зелёный	green
		жёлто-зелёный	green_yellow
		ярко-розовый	hot_pink
		лайм	lime
		оранжевый	orange
		розовый	pink
		фиолетовый	purple
		красный	red
		WG подсветка	wg_blur
		WG вражеский	wg_enemy
		WG дружественный	wg_friend
		WG взводный	wg_squad
		жёлтый	yellow

Обновлено в версии 3.08
	Добавлен обработчик индикации для сторонних модов
	1. запуск индикации из другого мода:
		if hasattr(BigWorld.player(), 'autoaim_extended'):
                	BigWorld.player().autoaim_extended.start(target)
	2. остановка индиакции из другого мода:
		if hasattr(BigWorld.player(), 'autoaim_extended'):
                	BigWorld.player().autoaim_extended.stop()
	3. текущая цель в индикации:
		if hasattr(BigWorld.player(), 'autoaim_extended'):
                	print BigWorld.player().autoaim_extended.target
