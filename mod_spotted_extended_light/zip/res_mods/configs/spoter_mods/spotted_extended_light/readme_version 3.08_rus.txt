Текущая версия 3.08 для 0.9.10 (18/11/2015)

Описание:
Мод показывает сообщения о засвете, ассисте по вашим разведданым и сбитым вами тракам, и убийстве по вашему засвету.
В тексте сообщения можно указать макросы с иконкой противника, именем и названием танка, настраивается в конфиге.
